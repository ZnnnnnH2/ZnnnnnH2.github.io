寄存器

63 31 15 7 意义 注
%rax %eax %ax %al 返回值,算术运算 返回值必须在 %rax 中

%rsi %esi %si %sil 第 2 个参数
%rdi %edi %di %dil 第 1 个参数

%r8 %r8d %r8w %r8b 第 5 个参数
%r9 %r9d %r9w %r9b 第 6 个参数

操作数指示符

类型 格式 操作数值 名称
立即数 $Imm Imm 立即数寻址
寄存器 $r_a$ $R[r_a]$ 寄存器寻址
存储器 Imm $M[Imm]$ 绝对寻址
存储器 $(r_a)$ $M[R[r_a]]$ 间接寻址
存储器 $Imm(r_b)$ $M[Imm+R[r_b]]$ 基址+偏移量寻址
存储器 $(r_b,r_i)$ $M[R[r_b]+R[r_i]]$ 变址寻址
存储器 $Imm(r_b,r_i)$ $M[Imm+R[r_b]+R[r_i]]$ 变址寻址
存储器 $(,r_i,s)$ $M[R[r_i]\cdot s]$ 比例变址寻址
存储器 $Imm(,r_i,s)$ $M[Imm+R[r_i]\cdot s]$ 比例变址寻址
存储器 $(r_b,r_i,s)$ $M[R[r_b]+R[r_i]\cdot s]$ 比例变址寻址
存储器 $Imm(r_b,r_i,s)$ $M[Imm+R[r_b]+R[r_i]\cdot s]$ 比例变址寻址

数据传送指令

MOV 类

指令 效果 描述 说明
mov S,D D ← S 将源操作数 S 的值传送到目标操作数 D 中
movb S,D D[7:0] ← S[7:0] 将源操作数 S 的最低字节传送到目标操作数 D 的最低字节中
movw S,D D[15:0] ← S[15:0] 将源操作数 S 的最低字传送到目标操作数 D 的最低字中
movl S,D D[31:0] ← S[31:0] 将源操作数 S 的最低双字传送到目标操作数 D 的最低双字中 以寄存器为目的时会将寄存器高 4 字节清零
movq S,D D[63:0] ← S[63:0] 将源操作数 S 的四字传送到目标操作数 D 中
movabsq I,R R ← I 将绝对的四字立即数传送到寄存器

常规的 movb 只能表示 32 位补码表示的立即数，而 movabsq 可以表示 64 位补码表示的立即数，但目的只能是寄存器。

MOVZ 和 MOVS

MOVZ 将剩余的高位全部清零，MOVS 将剩余的高位全部符号扩展。

指令 效果 描述
movzbw S,D D[15:0] ← ZeroExt(S[7:0]) 将字节零扩展到字
movzbl S,D D[31:0] ← ZeroExt(S[7:0]) 将字节零扩展到双字
movzbq S,D D[63:0] ← ZeroExt(S[7:0]) 将字节零扩展到四字
movzwl S,D D[31:0] ← ZeroExt(S[15:0]) 将字零扩展到双字
movzwq S,D D[63:0] ← ZeroExt(S[15:0]) 将字零扩展到四字
指令 效果 描述
movsbw S,D D[15:0] ← SignExt(S[7:0]) 将字节符号扩展到字
movsbl S,D D[31:0] ← SignExt(S[7:0]) 将字节符号扩展到双字
movsbq S,D D[63:0] ← SignExt(S[7:0]) 将字节符号扩展到四字
movswl S,D D[31:0] ← SignExt(S[15:0]) 将字符号扩展到双字
movswq S,D D[63:0] ← SignExt(S[15:0]) 将字符号扩展到四字
movslq S,D D[63:0] ← SignExt(S[31:0]) 将双字符号扩展到四字
cltq %rax ← SignExt(%eax) 将 %eax 符号扩展到 %rax

栈上操作

在 x86 中默认栈向低地址增长，入栈时栈指针减小，出栈时栈指针增大。

指令 效果 描述
pushq S %rsp ← %rsp - 8;
M[%rsp] ← S		 将操作数 S 入栈
popq D D ← M[%rsp];
%rsp ← %rsp + 8		 将栈顶元素出栈到操作数 D

算术和逻辑操作

指令 效果 描述
leaq S,D D ← &S 计算存储器操作数 S 的有效地址
inc D D ← D + 1 加 1
dec D D ← D - 1 减 1
neg D D ← -D 取负
not D D ← ~D 取补
add S,D D ← D + S 加
sub S,D D ← D - S 减
imul S,D D ← D * S 乘
xor S,D D ← D ^ S 异或
or S,D D ← D | S 或
and S,D D ← D & S 与
sal k,D D ← D << k 左移
shl k,D D ← D << k 左移(同 sal)
sar k,D D ← D >>_A k 算术右移
shr k,D D ← D >>_L k 逻辑右移

加载有效地址

leaq 指令用于计算存储器操作数的有效地址，并将其存储到目标操作数中。它不会访问内存，只进行地址计算。

特殊的算术操作

指令 效果 描述
imulq S %rdx:%rax ← S × %rax 有符号全乘法
mulq S %rdx:%rax ← S × %rax 无符号全乘法
cqto %rdx:%rax ← SignExt(%rax) 将 %rax 符号扩展到 %rdx:%rax
idivq S %rdx ← R[%rdx:%rax] mod S;
%rax ← R[%rdx:%rax] ÷ S		 有符号除法
divq S %rdx ← R[%rdx:%rax] mod S;
%rax ← R[%rdx:%rax] ÷ S		 无符号除法

控制流指令

条件码

标志位 描述
CF 进位标志
ZF 零标志
SF 符号标志
OF 溢出标志

leaq 指令不会影响任何条件码。

对于移位操作，只有最后一位移出的位会影响 CF 标志位，OF 为 0

比较指令 CMP TEST
指令 效果 描述
cmp S,D D - S 比较操作数 S 和 D
test S,D D & S 按位与操作数 S 和 D
访问条件码

SET

指令 同义名 效果 设置条件
sete D setz D ← ZF 相等/零
setne D setnz D ← ~ZF 不等/非零
sets D D ← SF 负数
setns D D ← ~SF 非负数
setg D setnle D ← (SF^OF)&ZF 大于(有符号>)
setge D setnl D ← ~(SF^OF) 大于等于(有符号>=)
setl D setnge D ← SF^OF 小于(有符号<)
setle D setng D ← (SF^OF)|ZF 小于等于(有符号<=)
seta D setnbe D ← CF&ZF 超过(无符号>)
setae D setnb D ← ~CF 超过或等于(无符号>=)
setb D setnae D ← CF 低于(无符号<)
setbe D setna D ← CF|ZF 低于或等于(无符号<=)

其目的操作数 D 只能是一个字节寄存器或内存字节。
