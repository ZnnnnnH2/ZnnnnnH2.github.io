php 伪协议

php://input

php://input 允许你读取原始的 POST 数据。推荐使用Burpsuite的 Repeater 功能来发送自定义的 HTTP 请求，而不是浏览器插件。

php://filter

php://filter 允许你对文件进行过滤操作，比如编码转换、Base64 编码等。
